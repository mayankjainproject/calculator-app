# calculator-app

Simple web-based calculator app.

Run locally:

1. Open `index.html` in your browser, or
2. Serve the folder and open http://localhost:8000

```bash
cd /workspaces/calculator-app
python3 -m http.server 8000
```

Files added:

- `index.html` — main app UI
- `style.css` — styles
- `app.js` — interaction logic
